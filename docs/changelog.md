# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - Initial Release

- Multi-voice TTS for Discourse threads
- OpenAI, Bark, and Piper TTS engine support
- Interactive TUI player with read-along highlighting
- Start from specific post number via URL or `-s` flag
- Web UI (experimental) with Web Speech API
- Pronunciation fixes for technical terms
- Audio caching for interrupted generations
